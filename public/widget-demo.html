Peluquería Estudio Norte

Página de ejemplo de un sitio de terceros que embebe el widget

Nuestros servicios

Este HTML simula el sitio propio de un negocio. El recuadro de al lado es el widget de Agendaflex montado con un simple <script>: 

<div id="agendaflex"></div>
<script src="https://tu-dominio/widget/agendaflex-widget.js" defer></script>
<script>
  window.addEventListener('load', function () {
    Agendaflex.mount('#agendaflex', {
      tenant: 'estudio-norte',
      publicKey: 'pk_...',
      apiBase: 'https://tu-dominio',
    });
  });
</script>

Faltan parámetros. Abrí esta página como /widget-demo.html?tenant=SLUG&key=pk_... (o usá /widget-demo en local, que completa los datos de la empresa demo).
